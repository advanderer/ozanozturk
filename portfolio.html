So far, I have only worked on one video game project during my first Game Development course at university. Be sure to check back later for more projects and download links once they are finished!

IMPACT EVENT

Impact Event is a 2D space shooter made with Unity 4.6.

The goal of the game is to destroy the approaching asteroid before it hits the planet. 

The game requires players to be careful and multitask, as not only the asteroid is covered with a revolving layer of meteorids that serve as a shield but there's also an incoming meteor storm that can cause significant damage to the planet.
